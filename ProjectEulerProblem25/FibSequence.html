1000-digit Fibonacci number

Built in is a while loop that will iterate through the Fibonacci terms until it finds a term with a value containing >= 1000 digits. This is a massive number. I, unfortunately, learned that javascript cannot handle this large of an integer. The loop will time out once it reaches an infinite term value at iteration 1477. Each iteration is logged in the console for you to review. If javascript was capable of providing the solution it would appear below this paragraph. The if statement built into the while loop would have wrote to the paragraph element with the id of "fib".

Since Javascript could not support please see FibSequence.java for the solution. Thanks to the BigInteger class I was able to solve this problem. Myt thought process for my code is provided in the .java file.
